# Pre requirement

## 建议提前自学

- Git [doc](https://git-scm.com/docs/gittutorial) [learn git branching](https://learngitbranching.js.org/)  
  掌握Git原理，能够使用一些基础功能
- Github [github](https://github.com/)  
  注册一个账户，尝试创建一个项目并与git工具结合往上push一些东西
- Docker [get-start](https://www.docker.com/get-started)  
  安装完环境，并学会启动一个debian镜像  
  尝试构建一个docker-compose，包含一个mysql数据库，和一个debian镜像，能够在debian镜像中访问到mysql数据库
- Mirai [project](https://github.com/mamoe/mirai)  
  尝试了解一下Mirai生态，并成功启动一个实例，登陆上你的QQ号

## 必须掌握的

- 至少拥有一个有以下环境的Unix-Like系统
  - Python3 (必须得是3！！！！！！！！)
  - Git
  - Docker
  - VSCode
- B站账号，用来给我投币

## If Windows

当然如果你喜欢折腾，也可以使用Windows，你需要
- WSL1/2
- VSCode
- Docker Desktop
- 准备随时修自电脑
